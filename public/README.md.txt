Introduction

This is a PHP-based Employee Employer Management System that allows users to manage employers and employees.

Features
________________________

Add, Edit, Delete Employers
Add, Edit, Delete Employees
Search Employees by Employer
User-friendly UI with Bootstrap

Requirements
________________________

Before running the application, make sure you have the following installed:

	- PHP 7.4 or later

	- MySQL or MariaDB

	- Apache or Nginx Web Server

	- Composer (for dependency management)

	- PHPUnit (for running tests)

Setup Database
__________________________

Import the SQL file (database.sql) into your MySQL database.
Update the database connection details in db_conn.php


	$servername = "localhost";
	$username = "root";
	$password = "";
	$database = "your_database_name";

	$conn = new mysqli($servername, $username, $password, $database);

Run The FIles
__________________________

Run the Server and
Put the Project Files in you www Folder
